# 项目进展

## 当前状态

- 项目名称已统一为 `MindAgent`
- Java 包路径已迁移为 `com.mindagent.agent`
- 默认数据库已切换为 MySQL
- 默认缓存已切换为 Redis
- Chroma 已接入本地 RAG 方案
- 默认 LLM 策略已调整为主聊天与 Spring AI 均默认走 Qwen 兼容云端接口
- 聊天记忆已升级为多会话 + MySQL 持久化 + Redis recent window 短期缓存
- 本地依赖统一通过根目录 `docker-compose.yml` 管理
- Docker 数据目录已整理到 `docker/mysql/data`、`docker/redis/data`、`docker/chroma/data`
- 后端本地运行目录已统一调整为 `backend/storage/`
- 文档与归档资料已整理到 `docs/`
- 数据库初始化脚本已落地到 `backend/src/main/resources/db/init_schema.sql`

## 已完成事项

- 完成项目从 `MindBridge` 到 `MindAgent` 的命名迁移
- 调整本地运行端口，避开已有 `campus-app`
- 配置 MySQL、Redis、Chroma 本地依赖
- 调整 nginx 反向代理到 `8082`
- 增加 MySQL 初始化脚本
- 在初始化脚本中加入默认用户数据
- 清理无必要的脚本目录
- 将知识库目录与仓库缓存目录从 `data` 迁移到 `storage`
- 新增真实会话接口：创建、列表、历史查询、删除
- 聊天上下文切换为 `session summary + recent window + current query + optional RAG`
- 增加会话摘要表、压缩日志表与 recent cache 回源/回填逻辑
- 前端静态页已支持新建/切换/删除会话与查看会话历史
