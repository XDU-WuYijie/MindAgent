MindAgent 管理入口

单独的登录页面

这里负责登录与身份切换。管理员进入知识库管理页，老师进入预约管理页，普通用户进入聊天与预约页面。

  • 管理员页专注知识库管理，数据源以 MySQL 为准。
  • Chroma 只承担向量检索，不作为管理真相源。
  • 登录后自动保存访问令牌，方便切换页面。

登录

等待登录
默认管理员账号通常为 `admin / admin123`，老师账号为 `teacher / teacher123`，普通用户账号为 `user / user123`。如果你改过初始化数据，按实际账号登录即可。
登录后跳转
Admin / Teacher / User
管理员进入 `admin.html`；老师进入 `teacher.html`；普通用户进入 `index.html`。
